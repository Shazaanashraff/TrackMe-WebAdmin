# Web Admin E2E Plan (Playwright)

## Super Admin
- Login -> dashboard cards load
- Managers CRUD (create, edit, deactivate)
- Bus requests: approve/reject -> audit log updated
- Operations: edit bus service type + booking status
- Routes: create and edit stops

## Manager
- Login -> manager dashboard
- Request bus account -> status appears in requests
- Edit bus details -> changes persist
- Tracking page shows live location history
